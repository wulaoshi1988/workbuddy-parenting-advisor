# 🍼 育儿顾问 — WorkBuddy Parenting Advisor

基于 **33本顶级育儿书** 的深度蒸馏知识，专为 WorkBuddy 设计的育儿顾问技能。

## 📚 知识来源

33本育儿书，7个Agent并发，约1000万tokens深度学习，三轮认知压缩。
按心理学/教育学理论根源重新分为 **8大学派** + 场景速查 + 跨派共识。

## 🏫 8大学派

| 学派 | 核心命题 | 代表书 |
|------|----------|--------|
| 园丁派 | 儿童有内在节奏，父母准备环境 | 《童年的秘密》《园丁与木匠》 |
| 人本主义沟通派 | 先处理感受再处理行为 | 《如何说孩子才会听》《非暴力沟通》 |
| 阿德勒民主管教派 | 不当行为=寻求归属感 | 《正面管教》《孩子：挑战》 |
| 神经科学派 | 每次互动塑造神经回路 | 《全脑教养法》《父母的语言》 |
| 内驱力派 | 自驱力=自主×胜任×归属 | 《自驱型成长》《终身成长》 |
| 觉醒派 | 孩子是你未疗愈童年的镜子 | 《家庭的觉醒》《母爱的羁绊》 |
| 综合导览 | 多学派整合+中国本土实操 | 《好妈妈胜过好老师》 |
| 专题区 | 男孩/女孩/内向/青春期/父亲 | 《养育男孩》《爸爸军团》 |

## 📂 文件结构

```
parenting-advisor/
├── SKILL.md              # 技能主文件（8学派速查、工具方法速查）
├── README.md
└── references/
    ├── schools.md        # 8大学派详解（理论根源、核心洞察）
    ├── books.md          # 33本书速查（一句话精要+核心方法）
    ├── consensus.md      # 7条跨派共识（33本书都同意的真相）
    ├── debates.md        # 7大争议议题（各派立场+判断地图）
    └── scenarios.md      # 常见育儿场景速查（沟通/行为/学习/关系）
```

## 🔧 安装

在 WorkBuddy 中，将此仓库克隆到 skills 目录：

```bash
git clone https://github.com/wulaoshi1988/workbuddy-parenting-advisor.git ~/.workbuddy/skills/parenting-advisor
```

然后在 WorkBuddy 对话中直接提问育儿问题即可自动触发。

## 💡 使用场景

- 孩子写作业拖拉、不爱学习
- 亲子沟通冲突、吼完孩子后悔
- 孩子发脾气、哭闹不止
- 行为问题（打人、撒谎、对着干）
- 不同阶段的养育困惑（男孩/女孩/内向/青春期）
